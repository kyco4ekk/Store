Store — консольное приложение для учёта склада
Приложение реализует простой функционал складской системы.
Проект выполнен на C# с использованием MS SQL Server.

Основные возможности:
1. Управление товарами
   - добавление нового товара
   - изменение названия товара
   - удаление товара
   - просмотр списка всех товаров
2. Учёт прихода
   - добавление прихода товара
   - изменение записи прихода 
   - удаление записи прихода
   - просмотр прихода за выбранный период
3. Учёт расхода
   - добавление расхода товара
   - изменение записи расхода
   - удаление записи расхода
   - просмотр расхода за выбранный период
4. Остатки на складе
   - автоматический расчёт остатка
   - вывод остатков по всем товарам
